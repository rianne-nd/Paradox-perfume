## ğŸƒ Quick Start

### View Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/rianne-nd/Paradox-perfume.git
   cd Paradox-perfume
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server for better experience:
   
   ```bash

   ```

## ğŸ¨ Customization

### Modifying Content

The website content is contained within `index.html`. Key sections to customize:

- **Hero Section** (line ~438): Update tagline and call-to-action
- **Collections** (line ~500): Modify fragrance offerings
- **About Section**: Update brand story
- **Product Details**: Edit pricing and descriptions

### Styling

The project uses Tailwind CSS (CDN). Custom styles are defined in:
- Inline `<style>` tags in the `<head>` section
- Tailwind configuration in `<script>` tags
- Custom CSS classes throughout the HTML
